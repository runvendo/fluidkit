# Security Policy

## Supported Versions

Only the latest published version of `fluidkit` is supported for security fixes.

## Reporting a Vulnerability

Report vulnerabilities privately through GitHub Security Advisories:

https://github.com/runvendo/fluidkit/security/advisories/new

If you already have a maintainer email from a private channel, you may use it
for first contact. Do not open a public issue for a suspected vulnerability.
